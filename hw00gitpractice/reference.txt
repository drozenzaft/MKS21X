Settings:
	git config --global username "Daniel Rozenzaft" – set username
	git config --global user.email drozenzaft@stuy.edu – set email
	git config --global core.editor emacs – set default text editor
	git config --list – displays a list of your settings

downloading:
	git clone (github's ssh link) - put the repo into your directory
	git pull (something) - copy a file from the repo

uploading:
	git add (something) - posts something on in the repo
	git commit (someting) - updates the something in git's eyes
	git push - reuploads the repo to github

This is some cool stuff!!!
